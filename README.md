# 🏥 Healthcare Data Cleaning & Analysis Project

## 📌 Project Overview
This project focuses on analyzing a healthcare dataset containing patient information such as age group, gender, department, treatment type, treatment cost, length of stay, and readmission risk. The goal is to clean the data, handle missing values, remove duplicates, and perform basic exploratory data analysis (EDA) to extract meaningful insights.

---

## 🎯 Objectives
- Clean and preprocess healthcare dataset
- Handle missing values and duplicates
- Perform exploratory data analysis (EDA)
- Identify patterns in patient data
- Visualize key insights using charts

---

## 🧰 Tools & Technologies Used
- Python 🐍
- Pandas
- Matplotlib
- Google Colab

---

- patient_id

- age_group

- treatment_cost

---

## 🧹 Data Cleaning Steps
- Removed duplicate records
- Handled missing values using mode/mean
- Cleaned column names
- Checked data types and structure

## 📈 Analysis Performed
- Gender distribution analysis
- Department-wise patient count
